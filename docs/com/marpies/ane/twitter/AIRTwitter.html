getHomeTimeline()method 
public static function getHomeTimeline(count:uint = 20, sinceID:String = null, maxID:String = null, excludeReplies:Boolean = false, callback:Function = null):void

Requests home timeline for logged in user.

Parameters

count:uint (default = 20) — Specifies the number of tweets to retrieve. Must be less than or equal to 200. The value of count is best thought of as a limit to the number of tweets to return because suspended or deleted content is removed after the count has been applied.
 
sinceID:String (default = null) — Returns results with an ID greater than (that is, more recent than) the specified ID. There are limits to the number of Tweets which can be accessed through the API. If the limit of Tweets has occurred since the sinceID, the sinceID will be forced to the oldest ID available.
 
maxID:String (default = null) — Returns results with an ID less than (that is, older than) or equal to the specified ID.
 
excludeReplies:Boolean (default = false) — Set to true to prevent replies from appearing in the returned list of tweets. You will receive up-to count tweets — this is because the count parameter retrieves that many tweets before filtering out replies.
 
callback:Function (default = null) — Function with signature callback(statuses:Vector.<AIRTwitterStatus>, errorMessage:String):void.
